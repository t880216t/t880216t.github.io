Win10和Ubuntu 17.04 64位双系统安装

发表于 2017-11-18 更新于 2022-05-09

新到手的电脑，是UEFI启动模式，默认win10，ubuntu一直没装上，我被恶心了半个月。

先说下环境：
1.华硕顽石4代，固态+机械硬盘，默认64位win10在固态。
2.ubuntu 17.04 64位 ISO镜像
3.8G垃圾U盘一个
4.启动盘生成软件Rufus
5.PC已经unable了安全启动模式(Secure Boot),不知道怎么关打对应厂商客服。

步骤：

第一步：制作刻录启动盘

预留出PC上需要装Ubuntu的盘，记住大小，进了安装程序他们的名字就不是C：D：E：了，而是sd/sdaxx啥，只能大概从大小和顺序看出来哪个是我们想放的地方。

第二步：制作刻录启动盘

使用支持UEFI启动的刻录工具刻录到U盘上，启动方式选择“UEFI+MBR”模式，其它默认，一路确认。

第三步：从USB启动

重启windows，按esc选择USB启动，会有多个，UEFxxx上面那个是我们的启动U盘。

第四步：安装系统

最好进入试用系统，设置啥的还是可控的。开始安装，到是否选择并存页面，选择“其它选项”，找到自己之前预留的空位置，改下分区：
1.给20G以上，设置挂载点位‘/’，文件系统为‘ext4’
2.其它空间,设置挂载点位‘/home’，文件系统为‘ext4’
3.默认已经会有个swap分区，不要动
4.最下面的安装引导器位置，点击下拉菜单选择‘windows boot manager‘。
装的过程中有联网,不然引导到程序可能会失败，但不要安装系统更新啥的，会很慢

第五步：完工

没有了，重启玩耍吧。后面坑还很多，再记吧。
