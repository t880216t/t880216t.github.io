微信自动进群小助手

发表于 2019-08-28 更新于 2022-05-09

当群人数达到100人后，用户无法再通过扫描群二维码加入，只能让用户先添加群内联系人微信，再由联系人把用户拉进来。这样，联系人员的私人微信会添加大量陌生人，给其带来不必要的打扰。

方法

利用python wxpy库，和一个已经在群里的微信，我们轻松实现自动进群的目的。
用户加群操作步骤：

  • 用户搜索机器人助手
  • 输入指定的验证信息如“进1群”，并添加好友
  • 自动加为好友并收到进群邀请
